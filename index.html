Architecture notes

Forecaster — GraphSAGE-style message passing over 8-zone graph. Input: lag-8 utilisation, neighbour aggregation, cyclic hour encoding. Output: μ±σ per 30-min step for 6-hour horizon.


Dispatcher — Two-stage stochastic MIP. Stage-1: binary {throttle, defer, discharge} decisions. Stage-2: recourse slack. Solved via LP relaxation + greedy rounding in <1ms.


Production path — Replace Ridge features with PyTorch Geometric GraphSAGE. Replace LP heuristic with Gurobi MILP (target: <200ms per solve cycle).
